# Koa2 API Starter

## Links
- [Git](http://github.com)

## Development
##### Requirements:
1. Node.js 7.6.*

##### Installing:
```sh
$ git clone -b [git-repo-url] api
$ cd api
$ npm i
$ npm run dev
```
Open link [http://localhost:3000](http://localhost:3000) and you should see json like this:
> { app: "Koa2 API Starter", error: "Not Found" }

##### Run tests
```sh
$ npm test                              // run all tests
```
